
#Generador de Tarjetas

> Note: Publicado en [Netlify][netlify] 


##Instrucciones:

Ingresamos a la carpeta:
```sh
cd Tarjeta
```
Se instalan las dependencias del proyecto:
```sh
yarn
```
Para visualizarla:
```sh
yarn dev
```
Para revisar linting:
```sh
yarn lint
```

[netlify]: <https://tarjetalomax.netlify.app>
